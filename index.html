Android-3d-engine

Android/Windows crossplatform 3D engine study

Features full OpenGL ES 2.0 support. Compiles under Android 2.3+ with JNI and Windows with Visual Studio 2008+. Includes working examples of Vertex and Pixel Lighting, Specular Reflections using different equations, and Shadow Mapping.

It also features a procedurally generated tubular racing track test. Download.
